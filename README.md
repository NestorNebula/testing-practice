# testing-practice

## Description

This project's goal is to practice Test-Driven Development by writing unit tests for different functions.

The project includes :

### Functions

- capitalize(_string_) : Return the given _string_ with the first letter capitalized
- reverseString(_string_) : Return the given _string_ reversed
- calculator : An object with four functions
  - add(_a_, _b_) : Return _a_ + _b_
  - subtract(_a_, _b_) : Return _a_ - _b_
  - divide(_a_, _b_) : Return _a_ / _b_
  - multiply(_a_, _b_) : Return _a_ \* _b_
- caesarCipher(_string_, _factor_) : Return a cipher version of the given _string_ based on the given shift _factor_
- analyzeArray(_array_) : Return an object containing informations (average, min, max and length) about the given _array_

### Tests

The [test file](./practice.test.js) contains tests for checking that the function works as expected.
